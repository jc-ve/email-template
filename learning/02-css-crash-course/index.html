CSS stands for Cascading Style Sheets!

CSS is used to add font styles, colors, backgrounds, spacing, and other design elements to your HTML.

h2 { color: blue;}

  1. "h2" is the selector.
  2. { } represents a CSS Style Declaration.
  3. "color" is the Property.
  4. "blue" is the Value.
